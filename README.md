# React TO DO manager

A simple ToDO Manager using react and Reactstrap.
You can add a task , delete a task .

<a href="https://reacttodomanager.herokuapp.com/">DEMO</a>

### UP & RUNNING
* `npm install`
* `npm start`
* visit `http://localhost:8080/`
